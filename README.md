V1.0    2016-05-13

1、自动检测串口数，并列举出来，如无串口则显示1-16的串口

2、发送区增加右键功能，可选择清除；点击、回车发送，双击可发送文件

3、点击扩展按键，可以循环发送16组不能的字符串数据

4、接收超过20万行自动保存程序目录下的log文件中，自动保存后会清空数据，并且重新接收。也可手动保存

5、接收时可选择是否带时间戳、是否需要转化成HEX显示、可暂停

6、对发送的数据可进行异或、累加和校验

7、可显示发送、接收的字节、当前窗口的总行数
